# 단어쑥 베타 테스트 가이드

**앱 이름:** 단어쑥 (DaneoSsuk)  
**버전:** 0.1.0 (MVP Beta)  
**테스트 날짜:** 2026년 5월  
**플랫폼:** iOS / Android (Expo Go)

---

## 개발 환경 실행 방법 (Replit)

1. Replit 프로젝트에서 **단어쑥 (expo)** 워크플로우를 실행합니다.
2. 워크플로우 패널에서 QR 코드 또는 URL이 표시될 때까지 기다립니다.
3. 아래 Expo Go 섹션으로 이어집니다.

---

## Expo Go로 테스트하는 방법

1. 휴대폰에서 [Expo Go](https://expo.dev/go)를 설치합니다.
2. Replit 워크플로우 로그에서 `exp://` 또는 HTTPS QR 코드를 찾습니다.
3. **Expo Go → "Enter URL manually"**에 주소를 입력하거나 QR 코드를 스캔합니다.
4. 앱이 로드될 때까지 기다립니다 (첫 빌드는 1–2분 소요될 수 있습니다).

> **iOS 참고:** Expo Go는 App Store에서 무료로 설치 가능합니다.  
> **Android 참고:** Google Play 또는 expo.dev/go에서 설치합니다.

## 주요 테스트 체크리스트

### 온보딩

- [ ] 앱 첫 실행 시 온보딩 화면 (4단계) 표시
- [ ] "다음" 버튼으로 각 단계 이동
- [ ] "건너뛰기" 버튼으로 온보딩 스킵
- [ ] 온보딩 완료 후 홈 화면으로 이동
- [ ] 앱 재시작 시 온보딩 스킵 (이미 완료됨)

### 홈 화면

- [ ] 스트릭, 오늘 학습 수, 저장됨 수 표시
- [ ] "카메라 스캔" 버튼 탭 → 카메라 화면으로 이동
- [ ] "오늘 복습" 버튼 → 복습 화면으로 이동
- [ ] 추천 단어 탭 → 단어 상세 화면으로 이동
- [ ] 하단 탭바 모든 탭 작동 확인

### 카메라 스캔 (데모 모드)

- [ ] 카메라 화면 열림
- [ ] "데모 스캔 시작" 버튼 탭 → 2초 후 단어 칩 표시
- [ ] 인식된 단어 칩 탭 → 단어 상세 화면으로 이동
- [ ] 데이터셋 미등록 단어 칩 (예: "perseverance") → 미등록 단어 화면
- [ ] 미등록 단어 화면에서 "커스텀 단어로 저장" → 모달 열림
- [ ] 모달에서 한국어 뜻 입력 → 저장 → 단어 상세 화면으로 이동

### 단어 상세

- [ ] 단어, 발음기호, 뜻, 예문, 암기 팁 표시
- [ ] TTS 버튼 탭 → 영어 발음 재생
- [ ] 속도 조절 (0.8x / 1.0x / 1.2x) 작동
- [ ] 북마크 버튼 → 저장됨 탭에 즉시 반영
- [ ] "암기 시작" → 암기 화면으로 이동
- [ ] "스펠링 연습" → 스펠링 화면으로 이동
- [ ] 커스텀 단어: "수정하기" → 편집 모달 열림 (기존 값 사전 입력)
- [ ] 커스텀 단어: "삭제하기" → 확인 Alert → 삭제 → 이전 화면 복귀

### 암기 (플래시카드)

- [ ] 플래시카드 앞면 (영어 단어) 표시
- [ ] 탭하면 뒤집기 애니메이션 (한국어 뜻 표시)
- [ ] "쉬워요", "어려워요", "기억 안 나요" 버튼 탭 → 다음 카드
- [ ] 완료 화면에서 "스펠링 연습" / "랜덤 퀴즈" / "나중에" 작동
- [ ] 복습 큐 모드 (`mode=review`): 오늘 복습 단어 전체 순환

### 스펠링 연습

- [ ] 알파벳 버튼 탭으로 스펠링 완성
- [ ] 정답 시 초록색 피드백 + 다음 단어
- [ ] 오답 시 빨간색 피드백 + 힌트

### 검색

- [ ] 영어/한국어로 단어 검색
- [ ] 결과 탭 → 단어 상세로 이동
- [ ] "저장됨" 탭 → 저장된 단어 목록
- [ ] "최근" 탭 → 최근 본 단어 목록
- [ ] 커스텀 단어 저장 후 "저장됨" 탭 즉시 반영

### 복습

- [ ] 오늘 복습 단어 목록 표시
- [ ] "복습 시작" → 암기 화면(복습 모드)으로 이동
- [ ] 복습 완료 후 보류 → 다음날 스케줄 업데이트
- [ ] 예정 복습 목록 (D+1, D+3, D+7...) 표시
- [ ] 복습 완료 상태 ("오늘 복습 완료!") 표시

### 교과서 학습

- [ ] 초등 / 중등 / 고등 레벨 선택
- [ ] 학년 → 단원 드릴다운
- [ ] 단원 내 단어 목록 탭 → 단어 상세
- [ ] 완료된 단원 체크 표시

### 프로필

- [ ] 스트릭, 레벨, 학습 통계 표시
- [ ] 학습 달력 표시
- [ ] 업적 목록 표시
- [ ] "개발자 검증" 버튼 → 디버그 화면으로 이동

### 개발자 검증 (디버그 화면)

- [ ] 체크리스트 전체 통과 여부 표시
- [ ] 데이터 일관성 검사 (고아 항목, 중복 등)
- [ ] 흐름 테스트 버튼들 작동
- [ ] 초기화 버튼 클릭 → 확인 Alert 표시 → 초기화 작동

## 알려진 제한사항

| 기능 | 상태 |
|------|------|
| OCR 텍스트 인식 | ❌ 데모 모드 (실제 카메라 인식 없음) |
| 백엔드 / Supabase | ❌ 미연결 (로컬 AsyncStorage만 사용) |
| AI 암기 팁 | ❌ 미연결 (데이터셋 내장 텍스트만 표시) |
| 결제 / 구독 | ❌ 미구현 |
| 로그인 / 회원가입 | ❌ 미연결 (게스트 모드만 작동) |
| 클라우드 동기화 | ❌ 미구현 |
| 푸시 알림 | ❌ 미구현 |

---

## 데모 / 가짜 기능

- **카메라 스캔:** 실제 촬영 없이 미리 정해진 단어 목록 반환
- **OCR 인식:** 2초 딜레이 후 고정 단어 칩 표시
- **로그인:** "Supabase 연결 후 사용 가능" 메시지 반환
- **AI 암기 팁:** 데이터셋에 사전 작성된 텍스트 표시 (AI 생성 아님)

---

## 데이터 저장 방식

- **저장소:** `AsyncStorage` (기기 로컬)
- **저장 키:** `dss:savedWords`, `dss:reviews`, `dss:customWords`, `dss:streak` 등
- **초기화:** 프로필 → 개발자 검증 → 데이터 초기화 섹션 사용
